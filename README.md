- ### 项目介绍

  - ### 目录结构

    ```
    TilltheEnd/
    ├── Application/     # 应用层代码（如主程序、逻辑流程等）
    ├── Camera/          # 相机/视角相关模块
    ├── Core/            # 核心模块（算法、工具类、基础设施等）
    ├── Engine/          # 引擎整体控制与调度模块
    ├── exe/             # 可执行程序输出目录
    ├── Game/            # 游戏逻辑代码
    ├── Interface/       # 接口定义与抽象（纯虚类、公共API等）
    ├── intermediate/    # 中间产物，如编译文件、临时文件等
    ├── library/         # 项目库文件（如 .lib/.dll 的存放处）
    ├── Physics/         # 物理相关模块
    ├── Renderer/        # 渲染模块
    ├── resource/        # 资源文件（如 Shader/图片/音频等）
    ├── thirdparty/      # 第三方依赖与库文件
    ├── Window/          # 窗口相关模块
    ├── README.md        # 项目说明文档
    └── TilltheEnd.sln   # VS Solution 解决方案文件
    ```
